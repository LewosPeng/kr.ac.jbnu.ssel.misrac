MISRA-C:2004  Rule  14.4:  (Required) The goto statement shall not be used.

Example Code: 



#include "misra.h"
#include "m2cmex.h"

extern S16 test_1404( void )
{
   goto mylabel;               /* MISRA Violation */

mylabel:

   return 1;
}

QAC messages that encompass this guideline:

2001 A 'goto' statement has been used.
