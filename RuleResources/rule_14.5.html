MISRA-C:2004  Rule  14.5:  (Required) The continue statement shall not be used.

Example Code: 



/* PRQA S 2981 ++ */

#include "misra.h"
#include "m2cmex.h"


extern S16 test_1405( void )
{
   S16 r = 0;
   S16 n = 0;

   for ( n = 0; n < 5; n++ )
   {
      if ( n > 2 )
      {
         continue;           /* MISRA Violation */
      }
      ++r;
   }

   r = 1;

   return r;
}

QAC messages that encompass this guideline:

0770 A 'continue' statement has been used.
